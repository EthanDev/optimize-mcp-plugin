description: List all available knowledge base namespaces
---

List the available namespaces in the knowledge base using the `list_namespaces` MCP tool.

**Instructions:**

1. Call `list_namespaces` with no arguments
2. Present the namespaces in a clean list showing:
   - Namespace name
   - Number of rules (if provided in the response)
   - Brief description of what each namespace covers (if available)
3. Suggest next steps: "Use `/optimize-mcp:search <query>` to search within a specific namespace, or `/optimize-mcp:export <namespace>` to export rules."
